Reporting the Stock Market Status  Title  Providing a tabular report of the stock market status  
Description  Displaying the current market status in a tabular format.  
Input  None  
Processing  The system updates the market status in real-time with every change provided through 
the stock market web service.  
Output  A table with columns for symbol, company name, highest price, lowest price, last price, 
previous price, change, percentage change, best bid, and best ask. Each row shows the 
status of a stock symbol.  
Error 
Handling  Display an appropriate error message if the market status is not displayed.
